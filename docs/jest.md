# `jest` script

Adds Jest test runner.

```shell
ft jest
```

## Help

```
ft jest [options]

    Install Jest

Options
    --jest-config-file        config file name, defaults to jest.config.js
    --jest-test-dir           main testing folder, defaults to 'src/__tests__'
    --jest-setup-filename     runtime setup file, defaults to 'setup.js'
    --jest-ts                 boolean flag, if set, will install TypeScript transform
    --dir-src                 source folder, defaults to 'src'
    --dir-dist                dist folder to be added to Git ignore, defaults to 'lib'
    -h, --help                show this output
```
